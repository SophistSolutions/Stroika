Menus

About the menus

#if qMacOS

Apple

 The Apple menu offers the 'About qAppName ' command, and Apple system commands.
  • About qAppName - Display a dialog describing the creators of qAppName and telling how to get more info.
#endif

File

 The File menu offers the following commands:
  • New

    • Creates a new document.
  • Open

    • Opens an existing document. #if qMacOS
  • Close

    • Close the frontmost open window. #endif
  • Save

    • Saves the document using the same file name.
  • Save As

    • Saves an opened document to a specified file name.
  • Save A Copy As

    • Saves the document to a specified file name without associating the current buffer with that file. #if qMacOS
  • Revert

    • Discard any changes since the last time the document was saved. #endif #if qMacOS
  • Print Setup

    • Sets printer parameters.
  • Print

    • Prints a document.
  • Print One

    • Prints one copy of a document, avoiding the Print Job Dialog. #elif qWindows
  • Print

    • Prints a document.
  • Print Setup

    • Selects a printer. #endif #if qMacOS
  • Quit

    • Quits qAppName #elif qWindows
  • Exit

    • Exits qAppName #endif

Edit

 The Edit menu offers the following commands:
  • Undo

    • Undo the previous editing operation.
  • Redo

    • Redo the previously undone command<
  • Cut

    • Deletes the currently selected text and moves it to the clipboard.
  • Copy

    • Copies the currently selected text and objects to the clipboard.
  • Paste

    • Pastes data from the clipboard into the document.
  • Clear

    • Deletes the currently selected text and objects.
  • Select All

    • Selects all the text in the document.
  • Find

    • Brings up a dialog specifying a search string and then searches the document.
  • Find Again

    • Search for the next occurence of the current "Find" string.
  • Enter 'Find' String

    • Make the currently selected text the "Find" string. #if qWindows
  • Insert New Object

    • Inserts and embeds an object, such as a chart or an equation.
  • Object

    • List object-specific commands for when the current selection is a single OLE object #endif

Format

 These are commands used to format the text.
  • Font Name

    • Apply the give named font to the current selection.
  • Font Style

    • Apply the given font style to the current selection.
  • Font Size

    • Apply the given font size to the current selection. #if qWindows
  • Choose Font Dialog

    • Bring up a font & size & style selection dialog, and then apply the choice to the current selection. #endif
#if qMacOS

Windows

 This is a list of the currently opened windows (documents). The one with the checkmark is currently frontmost, and active. Any window (document) on the list can be selected from the menu, and its associated document (window) will be activated. #endif #if qWindows

Options

 These commands allow configuration of some cosmetic qAppName features.
  • Toolbar

    • Toggle whether or not the toolbar is displayed.
  • Status Bar

    • Toggle whether or not the status bar is displayed.
#endif

Help

 These commands provide access to help and other information about qAppName .
  • Browse Help

    • View the top level "Help" document. #if qWindows
  • About qAppName

    • Display a dialog describing the creators of qAppName and telling how to get more info. #endif
