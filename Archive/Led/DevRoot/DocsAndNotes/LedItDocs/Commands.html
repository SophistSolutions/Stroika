Commands

The meaning of most of these commands is obvious. Many are documented here only for the sake of completeness. And often commands are best understood in the context of some feature. In those case, we mostly just refer back to the section describing that feature.

New command (File menu)

#if qMacOS Use this command to create a new empty document. #elif qWindows Use this command to create a new empty document. Since qAppName is an SDI application, only one window can be opened at a time. So if qAppName is already editing a document, that document will be closed before the new one can be created. #endif

Open command (File menu)

#if qMacOS Use this command to open an existing document in a new window. #elif qWindows Use this command to open an existing document. Since qAppName is an SDI application, only one window can be opened at a time. So if qAppName is already editing a document, that document will be closed before the new one can be opened. #endif

You can create new documents with the New command.

Undo (Edit menu)

Undo the last editing action. For more information, see Undo.

Redo (Edit menu)

Redo the last undone editing action. For more information, see Undo.

Cut (Edit menu)

Copy the current selection to the clipboard, and then delete it. See Clipboard for more information.

Copy (Edit menu)

Copy the current selection to the clipboard. See Clipboard for more information.

Paste (Edit menu)

Insert the current clipboard contents into the current text buffer, obliterating the current selection. See Clipboard for more information.

Clear (Edit menu)

Deletes the currently selected text, or other object.

Select All (Edit menu)

Select all the text (and other objects) in the buffer.

Find (Edit menu)

Bring up a dialog, and allow the user to type in a string to search for. Then search from the current selection to find the given search string.

Find Again (Edit menu)

Search again for the current search string (see Find Command).

Enter 'Find' String (Edit menu)

Set the current search string (see Find Command) to the text which is currently selected.
