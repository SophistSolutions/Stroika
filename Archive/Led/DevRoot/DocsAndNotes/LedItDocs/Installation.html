Installation and System Requirements

System Requirements

The qTargetOSName version of qAppName requires:

    #if qMacOS
  • System 9 or later

    • qAppName may work on earlier versions of the MacOS, but it has not been tested there. It has been tested on MacOS 9.2, as well as under MacOS X (10.2). Earlier versions of qAppName can be used if you require working on older versions of MacOS.

  • Carbon

    • As of version 3.1, qAppName is now distributed as a Carbon application. Carbon is builtin to MacOS X and can be optionally installed in earlier versions. I believe it is automatically installed in versions after Mac OS 8.6.

    Because of now being a Carbon application, qAppName now runs natively on MacOS X, and no longer requires the OS 9 compatability box to run.

  • PowerPC

    • qAppName application for Macintosh no longer supports the 68K. For 68K support, you must use qAppName 2.1 or earlier.

    #elif qWindows
  • Windows 95/98/ME/NT/XP/2000 (required)

    • qAppName is a 32-bit application, optimized for the Windows platform. It may also run on other windows systems supporting Win32 (e.g. Win32s), but it hasn"t been tested on any others. #endif

How to install qAppName :

#if qMacOS
This distribution package contains
  • qAppName qVersion
  • LedItDocs folder containing qAppName documentation

Simply copy the application and the documentation where-ever you like on your disk. #elif qWindows

qAppName can be installed either manually, or via an automatic self-extracting installer. Most people will choose to install qAppName using the installer.

Running the qAppName installer will prompt you as to where you want to put your copy of qAppName. qAppName does not require and special DLLs to be installed (except that the installer itself is based on the Microsoft "Windows Installer" technology which will install itself on your system early in the installation process if you don"t already have it installed).

Things the installer will install:
  • qAppName editor ( #if qBuildForLedLineIt LedLineIt.exe #else LedIt.exe #endif )
  • LedItDocs folder containing qAppName documentation
#endif
