Scripting NetLedIt! 3.0 demo

Below you should see a text edit control. This is the NetLedIt! Netscape plugin. Commands are accessed via right mouse clicking.

Also, you can use the buttons below to test out NetLedIt!'s automation support. Say "View Source" in your web browser to see how simple it is to script NetLedIt!.


Notes:

  • Clicking the VertSBar/HorzSBar buttons toggle the tri-state property HasVerticalScrollbar/HasHorizontalScrollbar property. The three values are Yes, No, and IfNeeded
  • WrapToWindow toggles whether we wrap the given text to the controls width, or to the width of a page of text (useful in conjuntion with horizontal scrollbar property)
